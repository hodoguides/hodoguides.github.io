Conseils
Japon
eSIM ou pocket Wi-Fi recommandés (couverture excellente).
JR Pass utile seulement si vous faites plusieurs longs trajets (Tokyo↔Kyoto/Osaka, etc.).
Nombreux lieux refusent les batteries >100Wh dans certains musées/salles ; privilégier 10–20 000 mAh.
Beaucoup d’endroits restent cash-friendly ; avoir quelques yens même si le sans-contact progresse.
Consulte la carte: /japan/
Chine
Visa requis pour la plupart des nationalités (vérifie exemptions/visa transit 72/144 h).
Accès limité (Great Firewall) : pense à une solution d’accès approuvée avant le départ.
Avion : batteries externes en cabine uniquement, ≤100 Wh sans autorisation.
Paiement mobile omniprésent (WeChat/Alipay) ; cartes int’l parfois refusées.
Corée du Sud
Recharge T-money pour métro/bus partout, aussi valable dans certaines boutiques.
Clim forte l’été : apporte une couche légère pour intérieurs/métros.
Pharmacies très aidantes mais fermées tard ; pense aux basiques (pansements, médocs courants).
